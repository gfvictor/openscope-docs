# Rakudash Documentation Wiki

## About

Official multilanguage documentation and interactive knowledge base for OpenScope™ B2B ecosystem, detailing every
feature and process of its integrated platforms.

## Project Overview

This repository powers the technical and operational documentation for OpenScope™'s core platforms, including
**Rakudash**, **EnTori** and **WanSpot**. Built with modern web standards to ensure zero-latency navigation and seamless
localization.

## Tech Stack

- **Framework**: Next.js (App Router)
- **Language**: TypeScript, MDX (Markdown JSX)
- **Styling**: Tailwind CSS
- **Engine**: Fumadocs
- **Search**: Orama (Local CSR)

## Local Development

To run the development server locally:

Clone this repository to your local machine, and run

```bash
npm ci
npm run dev
```

Open http://localhost:3000 with your browser to see the result
