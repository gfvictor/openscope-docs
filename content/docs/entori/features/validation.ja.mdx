# セキュリティとデータ検証

データへの信頼は人事システムの柱です。**EnTori**は、データが検証されるまでユーザーの入力を信用しません。

### これは何ですか？

候補者が入力するすべての文字を検査する自動ブロックおよびアラートシステム（**Zod**と**React Hook Form**を採用）です。

### 何のためにあるのですか？

候補者が重要な情報（連絡先の電話番号など）を欠落したまま、または無効な形式（数字だけのメールアドレスや、未来の生年月日など）でフォームを送信するのを防ぐために機能します。

### どのように使われますか？

ユーザーがフォームに入力している間、バックグラウンドで瞬時に検証が行われます：

1. 候補者が「名前」の入力を忘れた場合、ドキュメントを送信できません。
2. 間違ったフィールドや不完全なフィールドがある状態で「応募を送信」ボタンをクリックすると、不足している内容を説明する明確なエラーメッセージ（候補者が選択した言語に翻訳されたもの）がフィールドの下に表示されます。
3. すべてのエラーが修正されるまで送信はブロックされます。

### 期待される結果

**100%のデータ整合性**を保証します。派遣会社は、電話番号が欠落していたり、ビザの種類が指定されていない候補者の登録を受け取ることは決してなくなり、やり直し作業が排除されます。
