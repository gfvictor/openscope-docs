# EnTori（オンラインフォーム）

### これは何ですか？

人事データの取得に特化したレスポンシブなWebアプリケーション（_PWA対応_）です。従来の紙のバインダーに代わり、非常に動的で安全なデジタルフォームとして、候補者のプロフェッショナルなプロフィールに関するすべての関連情報を取得します。

### 何のためにあるのですか？

EnToriの主な目的は、データ入力の標準化です。人事チームが筆記体を解読したり、印刷されたフォームの欠落データを処理したりする代わりに、**EnTori**は候補者が送信する情報の構造化を強制し、検証してデータベースですぐに利用できるようにします。

### どのように使われますか？

候補者は、自分のモバイルデバイスから公開リンク（または派遣会社が提供する_QRコード_）を通じてプラットフォームにアクセスします。個人データ、日本語レベル、専門資格の種類、そして過去の職歴があれば記入します。フォームを送信する前に、同意書にデジタル署名します。

### 期待される結果

送信されると、フォームは自動的に**Rakudash**の管理ダッシュボードに反映され、クリーンで翻訳されたタイポのない候補者プロフィールが生成されるため、人事の何時間もの手動スクリーニング作業を節約できます。このプロフィールから、管理者は候補者を従業員として採用したり、顧客企業に配置したり、履歴書をPDFやExcelとしてエクスポートしたりできるようになります。
