# **EnTori**（オンラインフォーム）

**EnTori**は派遣会社のデジタルな入り口です。以下の4つの基本的なメリットを提供し、採用プロセスを近代化します：

<Cards>
  <Card
    title="データの標準化"
    description="情報の構造化を強制し、判読できない文字やデータの欠落を排除します。"
  />
  <Card
    title="完全なデータ取得"
    description="個人データ、語学レベル、資格、職歴をシンプルなフローで収集します。"
  />
  <Card
    title="デジタル署名"
    description="候補者がスマートフォンから法的な同意書にデジタル署名できるようにします。"
  />
  <Card
    title="自動連携"
    description="クリーンに翻訳された候補者プロフィールをRakudashダッシュボードに即座に生成します。"
  />
</Cards>

---

### これは何ですか？

人事データの取得に特化したレスポンシブなWebアプリケーション（_PWA対応_）です。従来の紙のバインダーに代わり、非常に動的で安全なデジタルフォームとして機能します。候補者は、自分のモバイルデバイスから公開リンク（_またはQRコード_）を通じてプラットフォームにアクセスします。

### 何のためにあるのですか？

**EnTori**の主な目的は、人事部の何時間もの手動スクリーニング作業を節約することです。チームが印刷されたフォームを解読してシステムにすべて入力する代わりに、候補者自身がデータを入力して検証します。その結果、即座にデジタル履歴書が作成され、顧客企業への配置やPDF/Excelとしてのエクスポートが可能な状態になります。
