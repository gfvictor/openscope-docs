# EnTori (Formulário Online)

### O que é?

É uma aplicação web responsiva (_PWA-ready_) focada exclusivamente em captação de dados de Recursos Humanos. Ele substitui a antiga prancheta de papel por um formulário digital altamente dinâmico e seguro, onde são captadas todas as informações relevantes ao perfil profissional do candidato.

### Para que serve?

O objetivo principal do EnTori é padronizar a entrada de dados. Em vez de a equipe de RH decifrar letras cursivas ou lidar com dados faltantes em formulários impressos, o **EnTori** força a estruturação de informações enviadas pelo candidato, validando e tornando essas informações prontas para o banco de dados.

### Como é usado?

O candidato acessa a plataforma através de um link público (ou _QR Code_ oferecido pela agência) pelo seu próprio aparelho móvel. Ele preenche seus dados pessoais, nível de japonês, tipos de certificações profissionais e, caso haja, experiências profissionais passadas. Antes de enviar o formulário, ele assina os termos de consentimento digitalmente.

### Resultado Esperado

Uma vez enviado, o formulário cai automaticamente no painel administrativo do **Rakudash**, gerando um perfil de candidato limpo, traduzido e sem erros de digitação, economizando horas de triagem manual do RH. A partir desse perfil, o currículo do candidato fica disponível para o gestor efetivá-lo como funcionário, alocar à empresa clientes, exportar seu currículo como PDF ou Excel, etc.
