# アクセスとセキュリティ

Rakudashは、派遣会社が管理する機密データを保護するために特別に構築された、制限された安全な環境です。

### どのような機能ですか？

これは認証制御および管理プロファイル管理システムです。スタッフのアカウントを外部のアクセスから隔離し、求職者や従業員（EnToriまたはWanSpotユーザー）が内部の操作を閲覧できないようにします。

### 何のためのものですか？

システムの整合性を保証し、不正アクセスを防止し、管理者が自身のセキュリティ資格情報や表示データを更新する自律性を提供するためです。

### どのように使用しますか？

- **システムへのログイン:** アクセスには、登録された社内メールアドレスと個人のパスワードを使用します。初期メールアドレスは常に総合管理者によって提供されます。
  <Callout title="セッションタイムアウトに関する注意事項:">
    情報セキュリティ上の理由から、長時間の無操作状態が続くと、Rakudashは自動的にセッションを終了します。作業中にログイン画面が表示された場合は、再度認証情報を入力して続行してください。
  </Callout>
- **パスワードの回復:** パスワードを忘れた場合は、ログイン画面にある回復オプションを使用してください。システムから社内メールアドレスに直接、安全で一時的なリセットリンクが送信されます。
- **マイプロフィール:** 技術サポートに頼ることなく基本的な情報を管理するには、パネル内で自分のユーザーをクリックしてプロフィールモーダルを開きます。そこから、**表示名**の更新と**パスワード**の変更が可能です。

### 期待される結果

安全で自律的かつ追跡可能なアクセスが確保され、Rakudashの管理操作が不正な閲覧や外部からの攻撃から常に保護されること。
