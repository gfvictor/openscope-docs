# 派遣会社の物件

### どのような機能ですか？

派遣会社が従業員の宿泊施設として提供する賃貸物件（アパート、一軒家など）のマッピングと管理に焦点を当てたダッシュボードです。

### 何のためのものですか？

不動産会社との直接契約に関する情報を一元化し、各物件の運営コスト（光熱費）を管理し、これらの物件を入居する従業員の賃貸契約に結び付けるためです。

### どのように使用しますか？

インターフェースには、以下の情報を含む運営物件のリストが表示されます：

- 物件名；
- 不動産会社に支払う家賃額；
- 物件タイプ（例：1DK、2LDKなど）；
- 都道府県および市区町村；
- 不動産会社との契約開始日および終了日。

物件をクリックすると、管理者は2つのタブに分かれた管理パネルにアクセスします：

1. **物件情報:**
   - **基本データ:**
     - 完全な住所；
     - 間取りタイプ（2DKなど）；
     - 不動産契約の有効期間（開始および終了）；
     - 自由な運営メモのためのスペース。
   - **従業員契約:**
     - 入居している従業員との賃貸契約の詳細（物件が占有されている場合に表示されます）。

2. **物件管理:**
   - **追加コスト:**
     - 物件の運営経費（電気、ガス、水道など）を統合するためのインターフェース。
   - **不動産データ:**
     - 不動産会社との直接契約に関する登録情報および商業上の連絡先。

### 期待される結果

非の打ちどころのない住宅組織化。派遣会社は賃貸資産を完全に管理し、不動産会社との契約期限を監視し、入居する従業員へのコストの適切な割り当てを保証します。
