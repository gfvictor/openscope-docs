# エクスポート

### どのような機能ですか？

従業員の記録と現在の配属に関する表形式のデータ（データダンプ）を抽出するための堅牢なツールです。

### 何のためのものですか？

外部システム、サードパーティの会計事務所、または内部の管理監査への情報の戦略的な転送を可能にし、派遣会社の生データを扱う際の絶対的な柔軟性を保証するためです。

### どのように使用しますか？

インターフェースには、2つの運用カードに割り当てられたセグメント化された抽出オプションが表示されます：

1. **配属エクスポート（企業別）:**
   - 勤務先の企業によってパラメータ化された従業員リストの抽出。
   - `リストをエクスポート`アクションをトリガーすると、以下のオプションを持つ設定モーダルが開きます：
     - 特定の派遣会社オフィスによるフィルタリング；
     - 対象となる顧客企業の選択、またはグローバル抽出（バッチ内の各顧客の個別のスプレッドシートを生成します）；
     - エクスポートの希望する有効期間の区切り；
     - 契約ステータスフィルター（アクティブな従業員に限定するか、退職者を含む完全な履歴を含めるか）。

2. **グローバルエクスポート（全従業員）:**
   - 配属の有無にかかわらず、派遣会社に登録されている従業員のデータベース全体の大量抽出。
   - `リストをエクスポート`アクションをトリガーするには、管理者がモーダルで集計期間を定義するだけで済みます。
   - 結果として、企業の完全な人口動態および契約上の国勢調査を含む統合ファイルが生成されます。

<Callout title="ファイル標準:">
  市場の会計ソフトウェアとの互換性および直接統合を目指して、すべてのエクスポートルーチンはMicrosoft
  Excel（.xlsx）形式でのみレポートをパッケージ化して提供します。
</Callout>

### 期待される結果

制限のないデータの機動性。派遣会社は企業情報の主権を維持し、タイムリーな意思決定と説明責任に不可欠な複雑なレポートを、摩擦なしで迅速に抽出することができます。
