# 管理

### どのような機能ですか？

Rakudashの事務的および管理的な中枢であり、監査、内部プロセス、プロトコル、および派遣会社の企業としての「書類作成」を構成するすべての文書化に焦点を当てています。

### 何のためのものですか？

構造的な事業管理（物理的な支店や顧客企業の組織化）を、人の管理（人事と依頼）から分離するためです。B2Bの契約ルール、請求、およびオフィスの組織図が独立して安全に機能することを保証します。

### どのように使用しますか？

ダッシュボードは、派遣会社の構造的ツールへのアクセスを提供します。サイドバーのナビゲーションを通じて、管理者は**オフィス**（本社および支店）と**顧客企業**（工場や契約先）の登録およびビジネスルールにアクセスし、管理します。

### 期待される結果

非の打ちどころのない企業組織化。派遣会社は、物理的な業務と顧客関係（B2B）の完全な追跡可能性を持ち、請求、各部門のルール、および支店の階層が完璧に機能することを保証します。
