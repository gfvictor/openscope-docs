# オフィス

### どのような機能ですか？

本社およびすべての稼働中の支店を網羅する、派遣会社の運営部門の中央登録簿です。

### 何のためのものですか？

企業の物理的構造をマッピングし、責任を分散させ、誰がどの地域を管理するかを明確にし、どの顧客と従業員が各オフィスに属しているかを整理するためです。

### どのように使用しますか？

ダッシュボードには、オフィス名（例：豊橋本社、名古屋支店）、都道府県、責任者（管理者）、その役職、およびユニットの電話番号を含むモニタリングリストが表示されます。

リストからオフィスをクリックすると、3つのタブに分かれた管理パネルが開きます：

1. **オフィス情報:**
   - 完全な住所；
   - 電話およびFAX；
   - 直属の責任者のデータとその役職；
   - システムへの支店登録日。

2. **顧客企業:**
   - この支店にリンクされている顧客企業のリストと、稼働中の配属の総数；
   - 選択した顧客に配属されている従業員の氏名リスト。

3. **管理者:**
   - オフィスのデータにリンクされ、制限付きまたは完全なアクセス権を持つすべてのシステム管理者およびオペレーターのリスト。

### 期待される結果

絶対的なコントロールを伴う分散化。派遣会社は新しい支店を開設して事業を拡大し、中央集権的な監査と管理の能力を失うことなく、各拠点を独立して組織化された状態に保つことができます。
