# 有給休暇の管理

### どのような機能ですか？

従業員が取得した有給休暇（有休）日数の付与の計画と管理に特化した管理ダッシュボードです。

### 何のためのものですか？

休暇の権利取得サイクルを監視し、従業員と派遣会社が日本の労働法を完全に遵守して運営されることを保証するためです。有効期限の追跡を自動化し、法律で定められた期間内に義務付けられた日数を付与しなかったことによる派遣会社への制裁を防ぎます。

### どのように使用しますか？

マスターリストには、従業員の統合された休暇のシナリオが表示されます。公開されるデータには以下が含まれます：

- 従業員の氏名；
- 正式な採用日；
- 契約状況（配属中、待機中、または退職手続き中）；
- 現在配属されている顧客企業；
- 前回の休暇付与日；
- 次回の付与予測と迫り来る有効期限；
- これまでに付与された合計日数と使用された日数。

新しい休暇サイクルを挿入するために、管理者は以下のフローを実行します：

1. **手動スケジュール（新規登録）:**
   - 付与モーダルを開く；
   - 対象となる従業員の選択；
   - 正確な付与日の入力；
   - 勤続年数および労働法（有給休暇）の要件を条件として、加算される日数の定義。

### 期待される結果

絶対的なガバナンスとコンプライアンス。派遣会社は有効期限を決して逃さず、厳密に合法的な方法で福利厚生の分配を管理し、罰金や労働負債を未然に防ぎます。
