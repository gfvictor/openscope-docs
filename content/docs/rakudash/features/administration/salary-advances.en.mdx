# Salary Advances

### What is it?

An advanced module for financial control and salary retention regarding salary advances (vouchers) granted to employees. Unlike the Requests panel (where the application is received), the focus here is accounting audit.

### What is it for?

To manage the approval and payment flow of cash advances, offering historical search features and installment control, ensuring that every advanced amount is properly injected as a deduction in the employee's payroll.

### How is it used?

The interface displays requests for the current month, featuring an advanced search module for auditing the general history (granted and refused). The main list consolidates the following metrics:

- Requester's name;
- Client company of the current allocation;
- Requested monetary amount;
- Settled or already paid amount;
- Current request status;
- Issuance, approval/rejection, and payment dates;
- Number of installments required for settlement.

<Callout title="Agile Navigation:">
  Clicking on the requester's name immediately redirects the manager to the employee's registration
  profile, facilitating historical consultation prior to credit approval.
</Callout>

From this panel, the manager has access to two operational controls:

1. **Status Administration:**
   - State transition tool for approval, rejection, payment, or settlement of the request.

2. **Deductions Manager:**
   - Interface (modal) for the scheduled insertion of discounts, linking the advance installments directly to the employee's payroll.

### Expected Result

Mitigated financial risks. The agency tracks every penny advanced, guarantees staggered collections via installments, and automates the deduction in the monthly payment in a secure and auditable manner.
