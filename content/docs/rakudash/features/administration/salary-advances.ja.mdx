# 給与前借

### どのような機能ですか？

従業員に付与された給与前借りに関する財務管理と給与天引きのための高度なモジュールです。申請が受け付けられる依頼パネルとは異なり、ここでの焦点は会計監査です。

### 何のためのものですか？

前借りの承認および支払いフローを管理し、履歴検索機能と分割払い管理を提供することで、前借りしたすべての金額が従業員の給与の控除として適切に注入されることを保証するためです。

### どのように使用しますか？

インターフェースには当月の申請が表示され、一般的な履歴（承認および拒否）を監査するための高度な検索モジュールが備わっています。メインリストには以下の指標が統合されています：

- 申請者の氏名；
- 現在配属されている顧客企業；
- 申請された金額；
- 清算済または支払済の金額；
- 現在の申請ステータス；
- 発行日、承認/拒否日、および支払日；
- 清算に必要な分割払いの回数。

<Callout title="迅速なナビゲーション:">
  申請者の氏名をクリックすると、管理者は直ちに従業員の登録プロフィールにリダイレクトされ、信用の承認前に履歴の参照が容易になります。
</Callout>

このパネルから、管理者は2つの運用コントロールにアクセスできます：

1. **ステータス管理:**
   - 申請の承認、拒否、支払い、または清算のための状態遷移ツール。

2. **控除マネージャー:**
   - 割引をスケジュールして挿入するためのインターフェース（モーダル）であり、前借りの分割払いを従業員の給与計算に直接リンクします。

### 期待される結果

軽減された財務リスク。派遣会社は前借りされたすべてのペニーを追跡し、分割払いによる段階的な回収を保証し、安全で監査可能な方法で月々の支払いからの控除を自動化します。
