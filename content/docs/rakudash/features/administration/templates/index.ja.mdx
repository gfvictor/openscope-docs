# テンプレート

### どのような機能ですか？

システムのエクスポートおよび文書発行エンジンで使用されるファイルテンプレートを保存および設定するための一元化されたリポジトリです。

### 何のためのものですか？

派遣会社が発行する公式文書（履歴書、契約書、社内レポートなど）を標準化するためです。システムは、これらのExcelファイル（`.xlsx`）内の事前定義された変数をキャプチャし、従業員の実際のデータを注入することで、視覚的忠実度の高い最終文書を自律的に生成します。

### どのように使用しますか？

インターフェースは直接のファイルマネージャーとして機能します：

- **アップロードリポジトリ:**
  - 管理者は、構造的にフォーマットされたスプレッドシート（ヘッダー、企業のカラーパレット、ロゴを含む）をアップロードします；
  - データベースのフィールドに対応するタグおよび変数の構成（[注入タグ](templates/tags)のドキュメントを参照）。

- **内部管理:**
  - 履歴書およびプロフェッショナルプロファイルをエクスポートするための、現在アクティブなテンプレートの表示パネル；
  - 厳密に社内回覧用の官僚的なフォームおよび文書の管理。

### 期待される結果

企業の標準化と拡張性。派遣会社はExcelでの手動によるフォーム入力を軽減し、すべての外部文書がブランドの視覚基準に厳密に従い、人的ミスのない統合されたデータを使用することを保証します。
