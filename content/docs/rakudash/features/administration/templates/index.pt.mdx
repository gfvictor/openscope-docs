# Modelos

### O que é?

Um repositório centralizado para armazenamento e configuração de modelos de arquivos (templates), utilizados pelo motor de exportação e emissão documental do sistema.

### Para que serve?

Padronizar os documentos oficiais emitidos pela agência (como currículos, contratos e relatórios internos). O sistema capta as variáveis predefinidas nestes arquivos Excel (`.xlsx`) e injeta os dados reais dos colaboradores, gerando documentos finais de alta fidelidade visual de forma autônoma.

### Como é usado?

A interface opera como um gerenciador direto de arquivos:

- **Repositório de Upload:**
  - O gestor realiza o upload de planilhas estruturalmente formatadas (contendo cabeçalhos, paleta de cores corporativa, logotipos);
  - Configuração de _tags_ e variáveis correspondentes aos campos do banco de dados (consulte a documentação de [Tags de Injeção](templates/tags)).

- **Gestão Interna:**
  - Painel de visualização dos modelos atualmente ativos para a exportação de currículos (Rirekisho) e perfis profissionais;
  - Administração de formulários burocráticos e documentação de circulação estritamente interna.

### Resultado esperado

Padronização corporativa e escalabilidade. A agência mitiga o preenchimento manual de formulários no Excel, assegurando que toda a documentação externa siga rigorosamente o padrão visual da marca, com os dados consolidados e isentos de falhas humanas.
