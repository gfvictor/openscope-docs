# 注入タグ

### 概要

注入タグとは、管理者がExcelテンプレート（`.xlsx`）内に配置できる標準化された識別子（変数）です。エクスポート中、システムのエンジンはこれらのマーカーを検索し、対象となる従業員の実際のデータに自律的に置き換え、公式文書（履歴書や契約書など）の転記における人的ミスを軽減します。

### サポートされているタグの構造

置き換えを成功させるためには、二重波括弧の形式 `{{tag}}` を厳密に守ってタグをスプレッドシートに挿入する必要があります。

データベースによってマッピングされた公式の変数は、2つの主要なカテゴリに分かれています：

#### 1. 個人データ

- `{{avatar}}`: 従業員のプロフィール写真；
- `{{today}}`: 文書エクスポートの現在の日付；
- `{{name}}`: 氏名；
- `{{kana}}`: フリガナ/カタカナ形式の名前；
- `{{gender}}`: 性別；
- `{{birthdate}}`: 構造化された生年月日；
- `{{age}}`: リアルタイムで計算される現在の年齢；
- `{{nationality}}`: 出身国；
- `{{visa}}`: 在留資格の形式；
- `{{visaExpirationDate}}`: ビザの法的な有効期限；
- `{{japaneseLevel}}`: 日本語能力レベル（JLPTなど）；
- `{{phone}}`: 主要な連絡先電話番号；
- `{{email}}`: メールアドレス；
- `{{zipCode}}`: 郵便番号；
- `{{address}}`: 完全な現住所。

#### 2. 専門データ

職歴および学歴について、システムは反復的なインデックス形式を使用します。タグの末尾に、経験の年代順に対応する数字（`1`、`2`、`n`）を追加するだけです。例：`{{fromMonth1}}` は1月の月ではなく、最初に記録された経験の月を取得します。

**職歴:**

- `{{fromMonth1}}` から `{{fromMonthN}}`: 経験 _n_ の開始月；
- `{{toMonth1}}` から `{{toMonthN}}`: 経験 _n_ の終了月；
- `{{fromYear1}}` から `{{fromYearN}}`: 経験 _n_ の開始年；
- `{{toYear1}}` から `{{toYearN}}`: 経験 _n_ の終了年；
- `{{company1}}` から `{{companyN}}`: 経験 _n_ の企業名；
- `{{resignationReason1}}` から `{{resignationReasonN}}`: 経験 _n_ の退職理由。

**免許および資格:**

- `{{certificationMonth1}}` から `{{certificationMonthN}}`: 資格 _n_ の取得月；
- `{{certificationYear1}}` から `{{certificationYearN}}`: 資格 _n_ の取得年；
- `{{certificationType1}}` から `{{certificationTypeN}}`: 資格 _n_ のタイトルまたは種類。

<Callout title="運用の注意:">
  注入エンジンは大文字と小文字を厳密に区別し、空白にも敏感です（例：`{{ visa_expiration_date }}`
  ではなく `{{ visaExpirationDate }}`
  を使用してください）。企業データが正しく抽出されるよう、括弧内で提示されたとおりにタグを使用してください。
</Callout>
