# Dashboard

The dashboard is the first screen displayed after a successful login to Rakudash. It acts as the agency's visual command center.

### What is it?

An analytical Dashboard composed of numerical cards and dynamic charts that consolidate data from all other system modules.

### What is it for?

To offer managers a panoramic and immediate view of the company's health, alerting them to pending financial matters, expiring contracts, and employee turnover without the need to generate complex reports.

### How is it used?

The screen is divided into operational metrics structured in the following order:

- **Gender Comparison:** Displays the current number of male and female employees, along with the total number of active hires.

### Expected Result

Managers empowered with strategic information in real-time, allowing preventive rather than reactive actions (such as renewing contracts before they expire or preparing cash flow for requested salary advances).
