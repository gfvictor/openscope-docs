# ダッシュボード

ダッシュボードは、Rakudashへのログイン成功後に表示される最初の画面です。派遣会社の視覚的な司令塔として機能します。

### どのような機能ですか？

他のすべてのシステムモジュールからのデータを統合した、数値カードと動的なグラフで構成される分析ダッシュボードです。

### 何のためのものですか？

複雑なレポートを作成することなく、管理者に会社の健全性の全体的かつ即時的なビューを提供し、未処理の財務事項、契約の期限切れ、および従業員の離職について警告するためです。

### 期待される結果

管理者がリアルタイムの戦略的情報を把握できるようになり、後手ではなく予防的な行動（期限切れ前の契約更新や、申請された前払いのためのキャッシュフローの準備など）を可能にします。
