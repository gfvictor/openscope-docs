# 応募者

派遣会社の入り口です。このメニューには、御社で働くことに関心を示したすべての個人が集約されています。

### どのような機能ですか？

EnToriを介してデジタル履歴書に記入したものの、まだ面接や正式な承認プロセスを経ていないすべての個人のリストです。

### 何のためのものですか？

採用部門の「受信トレイ」として機能します。人事がプロフィールを評価し、面接のスケジュールを立てるために連絡を取り、応募者を採用するかどうかを決定する場所です。

### どのように使用しますか？

**応募者**メニューにアクセスすると、管理者は新規登録者の名前、写真、基本的な連絡先データを含む表を表示します。
特定の応募者をクリックすると、EnToriで記入された履歴書を含む完全なプロフィールが開きます。そこから、管理者は以下を行うことができます：

- **承認:** 応募者を「入社待ち」キューに移動します。
- **拒否:** 応募者を破棄します（アーカイブに移動します）。
- **メモを追加:** 電話や対面での面接に関する観察結果を記録します。

### 期待される結果

人材の損失を防ぎます。人事は毎日キューを整理でき、すべての関心のある人が迅速にフィードバックを受け取り、適切なプロフィールが迅速に採用に向けられるようにします。
