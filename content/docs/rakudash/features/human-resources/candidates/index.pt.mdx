# Candidatos

A porta de entrada da agência. Este menu concentra todos os indivíduos que demonstraram interesse em trabalhar com você.

### O que é?

Uma lista de todos os indivíduos que preencheram o currículo digital via EnTori, mas que ainda não passaram por nenhuma entrevista ou processo de aprovação formal.

### Para que serve?

Funciona como a "caixa de entrada" do setor de recrutamento. Serve para que o RH avalie os perfis, entre em contato para agendar entrevistas e tome a decisão de avançar ou não com o candidato.

### Como é usado?

Ao acessar o menu **Candidatos**, o gestor visualiza uma tabela com os nomes, fotos e dados básicos de contato dos recém-chegados.
Ao clicar em um candidato específico, abre-se o perfil completo contendo o currículo preenchido no EnTori. A partir daí, o gestor pode:

- **Aprovar:** Mover o candidato para a fila de "Aguardando Contratação".
- **Rejeitar:** Descartar o candidato (movendo-o para o arquivo morto).
- **Adicionar Notas:** Registrar observações sobre a entrevista telefônica ou presencial.

### Resultado esperado

Nenhum talento perdido. O RH consegue limpar a fila diariamente, garantindo que todos os interessados recebam um retorno rápido e os perfis adequados sejam rapidamente direcionados para a contratação.
