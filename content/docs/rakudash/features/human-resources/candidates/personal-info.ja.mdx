# 個人情報

応募者プロフィールの最初のタブには、提出された履歴書の完全なビューと、人事向けの意思決定ツールが表示されます。

### どのような機能ですか？

片側に個人の必須登録データを表示し、もう片側に資格履歴を表示するデュアルパネルであり、選考プロセスのコントロールボタンも集約されています。

### 何のためのものですか？

応募者の評価を集中管理するためです。ここで採用担当者は、プロフィールが求人要件を満たしているかを分析し、承認、拒否、または顧客（工場）に提示するためのフォーマット済み履歴書の生成を実行します。

### どのように使用しますか？

インターフェースは2つのメインカードに分かれています：

1. **個人情報:**
   - 写真、名前、フリガナを表示します。
   - 国籍、ビザの種類、ビザの有効期限、生年月日、配偶者の有無。
   - 日本語能力レベル（EnToriの基準に準拠）。
   - 完全な住所、電話番号、メールアドレス、銀行口座の詳細。
   - WanSpotアクセス認証情報の作成および承認に関する情報。
   - 応募ステータスを変更するボタン（承認または拒否）と、システムに登録されているテンプレートを使用して履歴書をExcelまたはPDFで生成およびエクスポートするボタン。

2. **ライセンスと証明書:**
   - EnToriでの登録時に応募者が報告したすべてのライセンスと証明書をリストします。
   - 管理者は新しい証明書を追加したり、既存の証明書を編集・削除したりして、資格履歴を常に最新の状態に保つことができます。

### 期待される結果

余分な書類作業のない効率的な選考。管理者はすべての情報が明確に配置されているため、ワンクリックで応募者を承認したり、顧客に送信するためのフォーマット済み履歴書を生成したりできます。
