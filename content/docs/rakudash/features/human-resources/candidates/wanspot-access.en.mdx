# WanSpot Access

The first digital touchpoint between the agency and the candidate, allowing them to access the service platform.

### What is it?

An administrative modal where the manager defines or changes the login credentials the individual will use to access the WanSpot application.

### What is it for?

To grant autonomy to the approved candidate or block access for rejected profiles. HR ensures that only authorized people can log into the application to interact with the agency.

### How is it used?

When clicking to manage access in the candidate's panel, the system opens a modal with the following controls:

- **Credentials:** The manager can manually set the username (login) and password the candidate should use.
- **Access Permission:** A simple checkbox that allows or revokes access to WanSpot. If unchecked, even if the user has the correct password and is already registered in the database, they will not be able to log into the application.

### Expected Result

Security and strict access control. The manager can unlock the WanSpot app for the candidate to start their admission process, or immediately revoke access if the hiring process does not move forward.
