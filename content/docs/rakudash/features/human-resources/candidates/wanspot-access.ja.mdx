# WanSpotアクセス

派遣会社と応募者の間の最初のデジタル接点であり、サービスプラットフォームへのアクセスを可能にします。

### どのような機能ですか？

管理者が、個人がWanSpotアプリケーションにアクセスするために使用するログイン認証情報を定義または変更する管理モーダルです。

### 何のためのものですか？

承認された応募者に自律性を与えるか、拒否されたプロフィールのアクセスをブロックするためです。人事は、許可された人物のみがアプリケーションにログインして派遣会社とやり取りできるようにします。

### どのように使用しますか？

応募者のパネルでアクセスを管理するためにクリックすると、システムは以下のコントロールを持つモーダルを開きます：

- **認証情報:** 管理者は、応募者が使用するユーザー名（ログイン）とパスワードを手動で設定できます。
- **アクセス許可:** WanSpotへのアクセスを許可または取り消すシンプルなチェックボックス。チェックを外すと、ユーザーが正しいパスワードを持ち、すでにデータベースに登録されていても、アプリケーションにログインできなくなります。

### 期待される結果

セキュリティと厳格なアクセス制御。管理者は、応募者が入社プロセスを開始できるようにWanSpotアプリのロックを解除したり、採用が進まない場合は即座にアクセスを取り消したりできます。
