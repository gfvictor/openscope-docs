# Acesso ao WanSpot

O primeiro ponto de contato digital entre a agência e o candidato, permitindo que ele acesse a plataforma de serviços.

### O que é?

Um modal administrativo onde o gestor define ou altera as credenciais de login que o indivíduo usará para acessar o aplicativo WanSpot.

### Para que serve?

Dar autonomia ao candidato aprovado ou bloquear o acesso de perfis rejeitados. O RH garante que apenas pessoas autorizadas consigam logar no aplicativo para interagir com a agência.

### Como é usado?

Ao clicar para gerenciar o acesso no painel do candidato, o sistema abre um modal com os seguintes controles:

- **Credenciais:** O gestor pode definir manualmente o nome de usuário (login) e a senha que o candidato deverá usar.
- **Permissão de Acesso:** Um _checkbox_ simples que permite ou revoga o acesso ao WanSpot. Se desmarcado, mesmo que o usuário tenha a senha correta e já esteja cadastrado no banco de dados, ele não conseguirá entrar no aplicativo.

### Resultado esperado

Segurança e controle de acesso rígido. O gestor pode liberar o aplicativo WanSpot para o candidato iniciar seu processo de admissão, ou revogar o acesso imediatamente caso a contratação não avance.
