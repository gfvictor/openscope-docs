# 配属状況

社員の派遣と職歴の完全な管理。

### どのような機能ですか？

社員の派遣（業務割り当て）、契約、保険、および顧客企業間の異動履歴に関するすべての情報を収集する詳細なダッシュボードです。

### 何のためのものですか？

社員がどこで、どのような契約条件で働いているか、採用されてからの派遣会社内での軌跡について管理者に完璧な情報を提供し続けます。

### どのように使用しますか？

ページには、分かれたいくつかの情報カードがあります：

1. **入社および配属状況:**
   - 配属タイプ；
   - 企業での受諾日；
   - 対象の配属先と部門；
   - 契約の開始日と終了日；
   - 顧客企業から派遣会社に支払われる金額。

2. **社会保険と雇用保険:**
   - 社会保険情報；
   - 雇用保険情報。

3. **履歴:**
   - 応募と顧客配属の履歴；
   - 配属先の異動履歴；
   - 退職履歴；
   - 一般的な職歴。

### 期待される結果

社員の職業および契約生活全体の360度のビューを提供し、派遣状況に関する迅速な決定と対応をいつでも可能にします。
