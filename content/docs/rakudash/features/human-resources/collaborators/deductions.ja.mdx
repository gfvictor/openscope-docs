# 控除

前払金と家賃の給与天引きの管理。

### どのような機能ですか？

最終的な給与計算で控除する必要がある費用や前払金を継続的に記録するための社員の財務ハブです。

### 何のためのものですか？

社員に金銭的な前払金を提供した際に派遣会社に損失や漏れがないようにし、社員が会社提供のアパートに住んでいる場合の家賃の自動徴収を容易にします。

### どのように使用しますか？

インターフェースは、以下の2つのセクションを通じて、社員の会社に対する負債に排他的に焦点を当てています：

1. **契約と家賃情報:**
   - 現在の社宅の特定；
   - 家賃の正確な月額控除額。

2. **前払金（給与前借り）:**
   - 前払金が提供された特定の日付；
   - 前払金の正確な金額；
   - 給与天引きのための追跡。

### 期待される結果

財務的支出の管理を中央に集約することで、給与計算の締め処理中の重大な人為的ミスや見落としを防ぎます。
