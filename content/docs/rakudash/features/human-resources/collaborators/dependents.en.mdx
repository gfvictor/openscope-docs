# Dependents Information

Database of family members for tax and social insurance purposes.

### What is it?

The complete textual record of the employee's legal dependents.

### What is it for?

Centralizes the employee's relatives' information, which is a primary and mandatory requirement for calculating annual tax deductions in Japan and for Shakai Hoken (Social Insurance) enrollment paperwork.

### How is it used?

The manager uses a clean table to record the data of each dependent:

1. **Dependents Registration:**
   - Full name;
   - Date of birth;
   - Relationship.

<Callout title="About document submission:">
  Following security guidelines, there is no function to attach documents or PDFs in this section.
  It serves strictly for database registration.
</Callout>

### Expected Result

Ensures that the accounting and Human Resources departments quickly have all the necessary family data to fill out declarations and trigger insurance policies.
