# Informações de Dependentes

Base de dados dos familiares para impostos e seguro social.

### O que é?

O registro textual completo dos dependentes legais do colaborador.

### Para que serve?

Centraliza as informações dos parentes do funcionário, que é um requisito primário e obrigatório para os cálculos de deduções fiscais anuais no Japão e para a papelada de inscrição no Shakai Hoken (Seguro Social).

### Como é usado?

O gestor utiliza uma tabela limpa para registrar os dados de cada dependente:

1. **Cadastro de Dependentes:**
   - Nome completo;
   - Data de nascimento;
   - Grau de parentesco.

<Callout title="Sobre o envio de documentos:">
  Conforme as diretrizes de segurança, não há função de anexar documentos ou PDFs nesta seção. Ela
  serve estritamente para o registro em banco de dados.
</Callout>

### Resultado Esperado

Assegura que o setor de contabilidade e de Recursos Humanos tenha rapidamente todos os dados necessários da família para preencher declarações e acionar seguros.
