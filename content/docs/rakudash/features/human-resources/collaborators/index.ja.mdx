# 社員

日常業務が行われる場所です。ここには、会社の効果的な労働力があります。

### どのような機能ですか？

稼働中の社員の公式かつ全体的な名簿です。すでに契約に署名し、現在顧客企業（配属先）で働いているすべての個人を表示します。

### 何のためのものですか？

派遣会社に収益をもたらす人々を管理するためのメインデータベースとして機能します。各社員がどこで働いているか、銀行口座の詳細、扶養家族、履歴をすばやく確認できます。

### どのように使用しますか？

**社員**メニューでは、表に稼働中の社員がリストされ、クイック検索ショートカットがあります。
社員のプロフィールを開くと、管理者は以下にアクセスできます：

1. **個人情報:**
   - 主要データ；
   - WanSpotへのアクセス情報。

2. **配属状況:**
   - 現在の工場；
   - 異動履歴；
   - 契約上の日付。

3. **休暇と有給休暇:**
   - 欠勤履歴；
   - 有給休暇の残日数。

4. **PDFドキュメント:**
   - 在留カードとパスポート；
   - 契約書と給与明細。

5. **控除:**
   - 付与された前払金；
   - 会社の寮の家賃控除。

6. **扶養家族情報:**
   - 家族構成の登録；
   - 税金と保険のデータ。

### 期待される結果

管理者が稼働中の社員に関するドキュメントや情報を3クリック以内で見つけられる司令塔となり、日常の問題解決を促進します。
