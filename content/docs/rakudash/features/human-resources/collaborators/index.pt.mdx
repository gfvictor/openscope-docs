# Colaboradores

Onde o dia a dia acontece. Aqui você encontra a força de trabalho efetiva da empresa.

### O que é?

O quadro geral e oficial de funcionários ativos. Exibe todos os indivíduos que já assinaram contrato e estão trabalhando no momento em uma empresa cliente (alocação destino).

### Para que serve?

Serve como o banco de dados principal para gerenciar as pessoas que geram receita para a agência. Permite consultar rapidamente onde cada pessoa está trabalhando, seus dados bancários, dependentes e histórico.

### Como é usado?

No menu **Colaboradores**, a tabela lista os funcionários ativos com atalhos de busca rápida.
Ao abrir o perfil de um colaborador, o gestor tem acesso a:

1. **Informação Pessoal:**
   - Dados principais;
   - Credenciais de acesso ao WanSpot.

2. **Situação de Alocação:**
   - Fábrica de atuação atual;
   - Histórico de transferências;
   - Datas contratuais.

3. **Folgas e Férias Remuneradas:**
   - Histórico de faltas;
   - Saldo de férias (Yukyu).

4. **Documentos PDF:**
   - Zairyu Card e Passaporte;
   - Contratos e holerites.

5. **Deduções:**
   - Adiantamentos (vales) concedidos;
   - Descontos de aluguel corporativo.

6. **Informações de Dependentes:**
   - Cadastro de familiares;
   - Dados para impostos e seguro.

### Resultado esperado

Uma central de comando onde o gestor encontra qualquer documento ou informação de um funcionário ativo em menos de três cliques, facilitando a resolução de problemas diários.
