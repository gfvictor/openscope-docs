# 休暇と有給休暇

有給休暇（有休）の管理と監査パネル。

### どのような機能ですか？

欠勤、そして主に日本の労働法によって保証されている有給休暇（有休）の残日数の管理に特化したセクションです。

### 何のためのものですか？

社員が権利を有する休暇日数とその使用履歴を綿密に追跡することで、派遣会社が労働法を完全に遵守していることを保証します。

### どのように使用しますか？

画面は二つのメインパネルに分かれています：

1. **休暇の管理:**
   - 追加日数の手動付与；
   - 残高の編集；
   - 日数の削除；
   - 雇用に基づくシステムによる自動計算。

2. **使用履歴:**
   - 正確な使用日；
   - 消化された日数。

### 期待される結果

休暇管理における完全な透明性を確保し、労働争議を回避し、社員が適切に権利を享受できるようにします。
