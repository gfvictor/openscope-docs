# PDFドキュメント

契約書、給与明細、および社員ファイルの公式リポジトリ。

### どのような機能ですか？

社員の生活に関して生成、エクスポート、またはアーカイブする必要があるドキュメント専用の、安全で制限されたデジタル保管庫です。

### 何のためのものですか？

事務的なドキュメント（雇用契約書や毎月の給与明細など）を整理し、安全に保つためです。デジタルゴミを防ぎ、企業のセキュリティを確保するために、ドキュメントの編集はRakudashの管理者のみに制限されています（社員はWanSpot経由で直接アップロードできません）。

### どのように使用しますか？

このエリアでは、管理者は以下にアクセスできます：

1. **エクスポート可能なドキュメント:**
   - 雇用契約書の生成；
   - 給与明細の表示とエクスポート。

2. **ドキュメントの追加:**
   - 管理者によるRakudash経由の公式PDFの手動アップロード；
   - WanSpot経由での社員による直接アップロードの完全ブロック。

### 期待される結果

デジタルゴミのない、100%公式で監査可能なデジタル文書群であり、書類の履歴を安全かつ整理された状態に保ちます。
