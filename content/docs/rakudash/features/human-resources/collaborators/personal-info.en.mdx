# Personal Information

Central identity panel for the active employee.

### What is it?

It is the central identity and credentials panel for the employee. The interface is identical to the Candidates phase panel, but now focused exclusively on the active agency employee.

### What is it for?

It serves to group all civil, contact, banking information, and WanSpot access in a single quick-reference location.

### How is it used?

The manager navigates through two main cards:

1. **Personal Information:**
   - Displays name, furigana, and photo;
   - Nationality;
   - Visa type and expiration;
   - Marital status;
   - Japanese level;
   - Bank details;
   - Address.

2. **WanSpot Credentials:**
   - Login definition;
   - Password definition;
   - App access control.

### Expected Result

Ensures immediate and standardized access to all sensitive employee data and credentials in just a few clicks.
