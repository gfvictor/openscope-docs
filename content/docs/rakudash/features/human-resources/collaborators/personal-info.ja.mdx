# 個人情報

稼働中の社員のIDおよび資格情報の中央パネル。

### どのような機能ですか？

社員のIDと資格情報の中央パネルです。インターフェースは応募者フェーズのパネルとまったく同じですが、派遣会社の稼働中の社員に排他的に焦点を当てています。

### 何のためのものですか？

すべての市民情報、連絡先、銀行情報、およびWanSpotへのアクセスを1つのクイックリファレンス場所にグループ化するために機能します。

### どのように使用しますか？

管理者は2つのメインカードを操作します：

1. **個人情報:**
   - 名前、フリガナ、写真の表示；
   - 国籍；
   - ビザの種類と有効期限；
   - 配偶者の有無；
   - 日本語レベル；
   - 銀行口座の詳細；
   - 住所。

2. **WanSpotの資格情報:**
   - ログインの定義；
   - パスワードの定義；
   - アプリのアクセス制御。

### 期待される結果

数回のクリックで、すべての機密性の高い社員データとアクセス権への即時かつ標準化されたアクセスを保証します。
