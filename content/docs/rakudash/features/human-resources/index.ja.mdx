# 人事

人事モジュールは派遣会社の運営の中心であり、登録時から最終的な退職に至るまでの各個人の道のりを追跡します。

### どのような機能ですか？

システムに登録されているすべての個人のプロフィールを整理し、社内ライフサイクルの現在のフェーズに従って分類する集中型パネルです。

### 何のためのものですか？

人事部が採用や異動のパイプラインを完全に制御できるようにするためです。これにより、プロセスの中で応募者が放置されたり、異動待ちの社員が忘れられたりするのを防ぎます。

### どのように使用しますか？

サイドメニューの人事セクションはプロセスパイプラインとして機能し、5つの異なる作業キューに分かれています。各段階の管理方法を詳しく理解するには、横にあるサブトピックをナビゲートしてください：

- **応募者**
- **社員**
- **入社待ち**
- **異動待ち**
- **退職待ち**

### 期待される結果

管理者が各プロセス段階に何人いるか、部署の業務を滞らせないためにどのような行動を取る必要があるかを正確に把握できる、スムーズでボトルネックのない人材管理を実現します。
