# 入社待ち

### どのような機能ですか？

派遣先企業での配属をすでに承諾しているものの、就業開始日を待っているスタッフ専用のモニタリングリストです。

### 何のために使われますか？

現在パイプライン上（「オンホールド」）にある今後の採用予定者を一元的に把握し、クライアント企業の承認から実際の就業初日までの間に新しいスタッフが放置されるのを防ぎます。

### どのように使用しますか？

インターフェースには、今後の各配属に関する詳細な個別パネルが表示されます。

1. **配属情報パネル:**
   - 配属承諾日；
   - スタッフの氏名；
   - 派遣先企業と配属部門；
   - 就業形態（フルタイム、パートタイムなど）；
   - 就業開始予定日；
   - スタッフへの支給給与と給与形態（時給、日給、または固定月給）；
   - 初期の配属を担当した管理者の氏名；
   - 配属を確定した管理者の氏名およびその日付；
   - 現在の配属ステータス。

2. **管理と編集:**
   - **ステータス変更:** パネル上のクイックコントロールボタンを使用して、管理者が配属ステータスを即座に変更できます；
   - **詳細編集:** スタッフの名前をクリックすると完全なプロフィール（「候補者」セクションで説明したものと同じもの）が開き、就業開始日前に必要な登録データや契約データを調整できます。

### 期待される結果

採用から実際の就業開始までの、組織的で摩擦のないスムーズな移行。管理者は数回のクリックで目前の契約を承認、保留、または却下でき、派遣会社の入社フローを完全にコントロールできるようになります。
