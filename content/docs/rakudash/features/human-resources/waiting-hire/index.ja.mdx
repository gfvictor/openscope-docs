# 入社待ち

事務的な移行段階です。彼らはすでに承認されていますが、公式な書類がまだ不足しています。

### どのような機能ですか？

面接にはすでに合格しているものの、正式な契約の発行や、どの工場（配属先）に送られるかの正確な決定をまだ待っている応募者のための専用キューです。

### 何のためのものですか？

承認された応募者が返答を待つ「リンボ」状態に陥るのを防ぎ、人事が事務手続き（契約への署名、健康診断など）を可能な限り迅速に完了できるようにするためです。

### どのように使用しますか？

**入社待ち**メニューに入ると、管理者は承認されたプロフィールを表示し、以下を行うことができます：

- **配属先**（その人が働く工場）を定義する。
- テンプレートモジュールを使用して、**雇用契約** と入社書類を生成および発行する。
- プロセスを完了としてマークする。これにより、個人は自動的に**社員**の公式リストに移動します。

### 期待される結果

入社の迅速化。応募者は承認された後に回答を何日も待つことはなく、派遣会社は彼らがすぐに価値を生み出し始めることを保証します。
