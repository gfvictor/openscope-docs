# 異動待ち

人材定着のための最も重要な瞬間：ある顧客から別の顧客への移動。

### どのような機能ですか？

異動（ある工場を離れて別の工場に入る）の過程にある稼働中の社員の専用リストです。

### 何のためのものですか？

契約間の「ギャップ」（待ち時間）を管理するためです。社員が給与を受け取らずに何週間も過ごし、サポートが不足しているために別の派遣会社を探し始めることがないようにするために不可欠です。

### どのように使用しますか？

管理者が現在の工場の契約を終了し、その社員が派遣会社に留まることを示すと、社員はこのリストに入ります。
このメニュー内で、人事は以下のために迅速に行動する必要があります：

- 適切な新しい**配属先** を見つける。
- 新しい工場の新しい雇用契約を発行する。
- 異動を完了し、社員を**社員**名簿内の通常のステータスに戻す。

### 期待される結果

契約間の待機期間を最小限に抑えることによる人材定着率の最適化。派遣会社は、スタッフを戦略的に再配置するための予測・監視能力を獲得し、就業先不在による自主退職のリスクを軽減します。
