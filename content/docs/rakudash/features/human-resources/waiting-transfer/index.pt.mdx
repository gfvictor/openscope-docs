# Aguardando Transferência

O momento mais crítico para a retenção de talentos: a mudança de um cliente para outro.

### O que é?

Uma lista dedicada aos colaboradores ativos que estão em processo de mudança de alocação (saindo de uma fábrica para entrar em outra).

### Para que serve?

Gerenciar o "hiato" (tempo de espera) entre contratos. É crucial para que o funcionário não fique semanas sem receber salário e acabe procurando outra agência por falta de assistência.

### Como é usado?

O funcionário cai nesta lista assim que o gestor finaliza o contrato da fábrica atual, mas sinaliza que o colaborador continuará na agência.
Dentro deste menu, o RH deve atuar rapidamente para:

- Encontrar uma nova **Alocação Destino** adequada.
- Emitir um novo contrato de trabalho para a nova fábrica.
- Finalizar a transferência, devolvendo o funcionário ao status normal dentro do quadro de **Colaboradores**.

### Resultado esperado

Redução drástica do _turnover_ (rotatividade) por insatisfação ou demora. A agência consegue monitorar quem está "na geladeira" e realocar esses profissionais antes que eles peçam as contas.
