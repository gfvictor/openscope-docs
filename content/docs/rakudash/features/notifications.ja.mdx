# 通知

Rakudashの通知システムは、派遣会社の業務をリアルタイムで監視し、重要なアクションやリクエストについて管理者に即座に通知することを保証します。

### どのような機能ですか？

ページを更新することなく、エコシステム内で発生しているすべてのことに管理者を接続し続ける、プッシュ通知サービスと統合されたリアルタイムメッセージングエンジンです。

### 何のためのものですか？

日常業務の追跡において、電話やテキストメッセージに依存しないようにするためです。従業員がWanSpotで申請を行ったときや、新規求職者がEnToriでの履歴書登録を完了したときに、即座に管理者に警告します。

### どのように使用しますか？

アラートセンターは、日常業務において受動的および能動的に機能します：

- **ベル:** 画面の上隅にあります。クリックすると、最近のアラートの履歴を表示できます。
- **リアルタイム:** 画面にRakudashを開いている場合、新しいアラートは煩わしいポップアップを生成しません。代わりに、F5キーを押したりタブを再読み込みしたりしなくても、システムは即座にベルの上に未読通知の数を示す赤いバッジを表示します。
- **モバイルのプッシュ通知:** （新規求職者の登録などの）重要な通知は携帯電話やデスクトップのロック画面に直接送信されるため、即座に行動を起こすことができます。
  <Callout title="必須のインストール（PWA）:">
    システムを閉じた状態でロック画面にアラートを受信するには、事前にRakudashをデバイスにアプリとしてインストールしておく必要があります。
  </Callout>

### 期待される結果

休暇申請、給与前払い申請、または新規採用の数秒後に行動できる、応答性の高い管理チームが実現し、会社のコミュニケーションサイクルを完璧に完了させることができます。
