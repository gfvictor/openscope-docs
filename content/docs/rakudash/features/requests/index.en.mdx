# Requests

### What is it?

The central module that concentrates all operational and financial requests (such as vacations, leaves, and cash advances) submitted by employees through the WanSpot app.

### What is it for?

Unlike Human Resources, which focuses on contracts and allocation, this module serves to separate and organize purely administrative and cash flow demands. It ensures that requests are not lost in informal conversations and that the agency can plan finances and work schedules in advance.

### How is it used?

The module's navigation is divided into specific subsections according to the type of request. Managers use the sidebar to access lists of:

- Leaves and Vacations
- Salary Advances

In each subsection, the manager can monitor the status, the requester, the request date, and the usage date, responding quickly to each application.

### Expected Result

Absolute operational governance. The agency resolves employees' daily needs in a structured and auditable manner, ensuring that the cash flow and client operations are never caught off guard.
