# 依頼

### どのような機能ですか？

WanSpotアプリを通じてスタッフから提出されたすべての運営上および財務上の申請（有給休暇、欠勤、給与前払いなど）を一元化するコアモジュールです。

### 何のためのものですか？

契約と配属に焦点を当てた人事モジュールとは異なり、このモジュールは純粋な管理およびキャッシュフローの要求を分離して整理するために機能します。申請が非公式な会話の中で失われないようにし、派遣会社が事前に財務と勤務シフトを計画できるようにします。

### どのように使用しますか？

モジュールのナビゲーションは、申請の種類に応じて特定のサブセクションに分かれています。管理者はサイドバーを使用して以下のリストにアクセスします：

- 有給・欠勤
- 給与前借

各サブセクションで、管理者はステータス、申請者、申請日、使用日を監視し、各要求に迅速に対応することができます。

### 期待される結果

絶対的な運営ガバナンス。派遣会社はスタッフの日常的なニーズを構造化された監査可能な方法で解決し、キャッシュフローと顧客企業の運営が決して不意を突かれないことを保証します。
