# 有給・欠勤

### どのような機能ですか？

スタッフが行ったすべての不在申請を表示、監視、および対応するための管理ダッシュボードです。

### 何のためのものですか？

休暇、欠勤、または遅刻の申請を一元化し、派遣会社と最終顧客がスタッフの不在に対して適切に計画を立てられるようにします。出勤履歴を常に最新の状態に保ち、シフトの穴を防ぎます。

### どのように使用しますか？

インターフェースには、すべての申請を含むコントロールリストが表示され、以下の情報が含まれます：

1. **申請の詳細:**
   - スタッフの氏名；
   - 配属先の顧客企業；
   - 申請された使用日；
   - 不在の種類（有給、欠勤、遅刻）；
   - 時間の量（午前、午後、または全日）；
   - 現在の申請ステータス；
   - 申請を評価した管理者の氏名および確認日。

2. **申請の管理:**
   - リストパネル上の直接ボタンを使用して、管理者は申請のステータスを即座に**承認**、**却下**、または**キャンセル**に変更できます。

### 期待される結果

組織化と予測可能性。派遣会社は不在申請を迅速かつ文書化された方法で解決し、スタッフに対する透明性と顧客企業に対する運営上の安全性を保証します。
