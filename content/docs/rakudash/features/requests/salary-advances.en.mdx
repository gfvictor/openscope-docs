# Salary Advances

### What is it?

A financial management module focused exclusively on salary advance requests (vouchers) made by employees.

### What is it for?

To ensure that employees' requests for financial support are processed in a fast, structured manner linked directly to the agency's cash flow, eliminating parallel manual controls and mitigating the risk of undue payments.

### How is it used?

Financial managers monitor a consolidated list of requests, displaying:

1. **Advance Details:**
   - Employee's name;
   - Client company where they are allocated;
   - Requested amount;
   - Date the request was made;
   - Current request status;
   - Name of the manager who evaluated the request and the respective confirmation date.

2. **Approval and Control:**
   - Similar to leaves, the manager can **Approve**, **Reject**, or **Cancel** the request using a quick action button directly on the list panel, instantly updating the request status.

### Expected Result

Financial security and agility. The employee receives the necessary support in a regulated manner, and the agency maintains a strict, auditable accounting record of all advanced amounts.
