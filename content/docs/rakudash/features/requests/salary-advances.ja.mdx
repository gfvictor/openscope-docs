# 給与前借

### どのような機能ですか？

スタッフによる給与前払いの申請に特化した財務管理モジュールです。

### 何のためのものですか？

スタッフからの資金援助の要求が、派遣会社のキャッシュフローに直接結びついた形で迅速かつ体系的に処理されることを保証し、並行する手動の管理を排除し、不適切な支払いのリスクを軽減するためです。

### どのように使用しますか？

財務管理者は、以下を表示する統合された申請リストを監視します：

1. **前払いの詳細:**
   - スタッフの氏名；
   - 配属先の顧客企業；
   - 申請額；
   - 申請が行われた日付；
   - 現在の申請ステータス；
   - 申請を評価した管理者の氏名および確認日。

2. **承認と管理:**
   - 休暇と同様に、管理者はリストパネル上のクイックアクションボタンを使用して、申請を**承認**、**却下**、または**キャンセル**し、即座に申請ステータスを更新できます。

### 期待される結果

財務上の安全性と迅速さ。スタッフは規制された方法で必要なサポートを迅速に受け取り、派遣会社はすべての前払い額に関する厳密で監査可能な会計記録を維持します。
