# 公式契約

派遣会社と従業員の間で合意された条件に対する、完全な透明性と無制限のアクセス。

### これは何ですか？

労働契約書だけでなく、不動産の賃貸契約書（派遣会社が寮を提供し、従業員が借りている場合）も一覧表示する、改ざん不可能なデジタルリポジトリです。

### 何のためにあるのですか？

従業員が紙のコピーを取りに派遣会社のオフィスに行かなければならないという依存や、合意内容に対する疑問をなくすためです。いつでも自分の労働条件と居住条件を明確に把握できるように機能します。

### どのように使われますか？

`/contracts`セクションでは、従業員は現在および過去の契約書を開くことができます。自身の時給、ボーナス、規定労働時間、家賃の条件や控除額を自主的に確認できます。

### 期待される結果

従業員は常に契約書を手元に置くことで、より安心し、大切にされていると感じます。派遣会社は法的遵守を保証し、給与や住居のルールに関する繰り返しの質問を大幅に減らすことができます。
