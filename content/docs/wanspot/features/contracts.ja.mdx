# 公式契約書

確立された労働条件および居住条件の継続的な透明性と監査可能性。

### どのような機能ですか？

従業員に関連付けられた雇用契約書と社宅賃貸借契約書の完全な履歴を保存する、変更不可能で読み取り専用のデジタルリポジトリです。

### 何のためのものですか？

雇用関係の法的文書へのアクセスを分散化するためです。誤った情報による労働負債のリスクを軽減し、物理的なコピーへの依存や派遣会社への直接の訪問を排除します。

### どのように使われますか？

**ファイル**メニューから**契約書**オプションを選択することで、従業員は現在および過去の契約書の保存場所にアクセスできます。このインターフェースにより、時給、ボーナス構造、合意された労働時間、社宅に適用される条件や控除などの重要なデータを自律的かつ即座に参照できます。

### 期待される結果

コンプライアンスと相互の法的安全性の保証。従業員は常に条件を手元に置くことで法的保護を得る一方、派遣会社は労働時間や住宅控除に関する質問に関連する運用サポートを大幅に削減します。
