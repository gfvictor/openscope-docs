# 個人ドキュメント

電子文書管理と企業アナウンスのための一元化されたポータル。

### どのような機能ですか？

派遣会社が組織的および運用上のファイルを提供する一方で、同時に従業員からの機密文書の収集を可能にする、双方向の文書管理システムです。

### 何のためのものですか？

非公式チャネル（WhatsAppなど）を通じた機密データのトラフィックを排除し、物理的な郵送コストを最適化するためです。源泉徴収票の構造化された配信と、パスポートや在留カードのコピーの安全な収集を可能にします。

### どのように使われますか？

人事部は、**Rakudash**を通じて従業員のプロフィールに（または全体配信を通じて）ファイルを直接アップロードして提供します。**WanSpot**を通じて、従業員は**ファイル**メニューに移動し、**ドキュメント**オプションを選択することで、企業のアナウンス、会社の指導資料、および関連する税務書類（源泉徴収票など）に即座にアクセスできます。

### 期待される結果

公式で安全かつ追跡可能なチャネルの確立。派遣会社は重要なコミュニケーションを管理された環境に統合し、データの損失から運用を保護し、従業員の機密情報を保護します。
