# ドキュメントハブ

公式な一方向のコミュニケーションと重要な文書の配布。

### これは何ですか？

派遣会社が従業員の閲覧とダウンロード専用に、重要なファイルおよび会社のポリシーを提供する安全なデジタルライブラリです。

### 何のためにあるのですか？

事務的な通知の公式チャネルとして機能するためです。派遣会社は、重要なお知らせや公式ドキュメント（源泉徴収票など）がWhatsAppのリスクなしに、安全かつプライベートに従業員に届くことを保証する必要があります。

### どのように使われますか？

人事部は、**Rakudash**経由で従業員のプロフィールに（または全体に）ファイルを直接アップロードします。従業員は**WanSpot**の`/documents`ルートにアクセスし、そこでお知らせを読んだり、会社の一般ファイルにアクセスしたり、税務ファイルを保存したりできます。

### 期待される結果

機密ファイルや一斉送信のお知らせが安全な環境でデジタル配信されることが保証され、会社のコミュニケーションが統一され、重要な情報が時間の経過とともに失われるのを防ぎます。
