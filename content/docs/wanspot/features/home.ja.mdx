# 初期ダッシュボード

従業員が**WanSpot**にログインすると、現在の状況を把握するためにナビゲートする必要はありません。メイン画面で必要なすべての情報を一目で確認できます。

### これは何ですか？

従業員の完全なプロフィールと、顧客企業への配属データを含むサマリーページです。

### 何のためにあるのですか？

派遣会社が正確なデータを保持していることを従業員に認識させるとともに、自身の現在の雇用状況に関する疑問をすばやく解決するために機能します。

### どのように使われますか？

ホームページを開くと、従業員は以下のデータにすぐにアクセスできます：

- **個人情報：** 氏名、生年月日と正確な年齢、現在の住所。
- **配属状況：** 現在働いている顧客企業の名前と、その工場での勤務開始日。
- **派遣会社での履歴：** 派遣会社への正式な入社日。
- **有給休暇：** 取得可能な有給休暇の正確な現在の残日数。

### 期待される結果

従業員は、派遣会社が自分の登録データと配属データを正確かつ最新に保っていることを確認して安心感を得ることができ、人事部は「有給休暇は何日残っていますか？」という毎日の質問を受けなくなります。
