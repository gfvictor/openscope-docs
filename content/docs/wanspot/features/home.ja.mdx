# 初期ダッシュボード

認証後すぐに、従業員の指標と契約ステータスへの統合されたアクセスを提供します。

### どのような機能ですか？

従業員に焦点を当てたエグゼクティブダッシュボードインターフェースであり、完全なプロフィール、現在の配置ステータス、および派遣会社での主要な指標を提示します。

### 何のためのものですか？

従業員の登録および雇用状況に関する絶対的な透明性を提供するためです。基本的なコミュニケーションの摩擦を排除し、人事部の介入なしに、従業員が自身のデータとステータスを監査できるようにします。

### どのように使われますか？

プラットフォームの認証後、従業員はダッシュボードで以下のデータに即座にアクセスできます。

- **個人情報：** システム内で更新された氏名、生年月日、住所。
- **配置情報：** 現在配置されている顧客企業と、その工場での業務開始日。
- **職歴：** 派遣会社への公式な入社日。
- **有給休暇（有休）：** 取得可能な有給休暇の統合および更新された残日数。

### 期待される結果

些細な問い合わせ（有休の残日数や住所の確認など）による人事部への連絡量が激減します。継続的かつ透明性のあるデータの可視化を通じて、従業員は派遣会社のコンプライアンスに対して完全な信頼を得ることができます。
