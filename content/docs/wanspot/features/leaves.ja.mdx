# 有給・欠勤の管理

勤怠管理のガバナンスと、欠勤（休み）および有給休暇（有休）の追跡可能性。

### どのような機能ですか？

予定された、または予期しない欠勤的の登録、分類、および承認ワークフローに焦点を当てたトランザクションモジュールです。

### 何のためのものですか？

非公式なコミュニケーションに起因する顧客工場との人員配置のずれを軽減するためです。一元化により、有給休暇（有休）および欠勤（休み）の申請が正式に記録され、情報がボトルネックやデータの損失なしに管理者に確実に届くようになります。

### どのように使われますか？

メインメニューの**有給・欠勤**タブを通じて、従業員は予定された欠勤日を入力し、欠勤の理由を分類します。申請はシステム内で即座に正式に記録され、派遣会社の管理者にタイムリーなアラートをトリガーします。これにより、顧客工場との人員再配置や有給休暇の支払いを適切に処理するための十分な時間が確保されます。

### 期待される結果

勤怠管理の完全な監査可能性。運用側は顧客工場での人員の再配置に対応する時間を確保でき、従業員は申請の変更不可能な記録を通じて法的安全性を獲得します。
