# 休暇と欠勤の管理

時間を記録し、従業員、派遣会社、および顧客工場間のコミュニケーションを正式なものにします。

### これは何ですか？

従業員が仕事に出席しない日を記録することに特化したスケジューリングおよび通知インターフェースです。

### 何のためにあるのですか？

断片的で非公式なコミュニケーションを排除するためです。従業員が休む必要がある場合（休み）や有給休暇を使用したい場合（有給）、単なる口頭の通知や単発のメッセージは失われやすく、工場との問題を引き起こします。

### どのように使われますか？

`/leaves`タブを通じて、従業員は休む必要がある日付を入力し、欠勤の理由を分類します。申請はシステム内で正式に記録され、派遣会社の管理者にタイムリーに警告を発するため、顧客工場に情報を伝えたり、有給休暇の支払いを処理したりできます。

### 期待される結果

申請はシステムに正式に文書化され、不当な欠勤から従業員を保護し、派遣会社に工場の人員配置を完璧に管理するために必要な時間と記録を与えます。
