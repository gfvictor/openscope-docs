# 通知とアラート

更新を確認するために常にアプリを開く必要なく、従業員に常に情報を提供するためのプロアクティブなコミュニケーション。

### これは何ですか？

**WanSpot**ダッシュボードに統合されたリアルタイムのメッセージングおよびアラートシステムであり、派遣会社との雇用関係に関連する重要なイベントについて従業員に通知します。

### 何のためにあるのですか？

従業員の不安や、人事サポートへの「前借りは承認されましたか？」「有給は許可されましたか？」といった過剰なメッセージをなくすためです。管理者が何らかのアクションを起こすとすぐに、システムが従業員に受動的に通知する役割を果たします。

### どのように使われますか？

通知は自動的にトリガーされ、従業員は以下に関するリアルタイムのアラートを受け取ります：

- **給与前払い：** 人事部によって前借りの申請が承認または拒否されたときの即時通知。
- **休暇と有給：** 欠勤通知または有給休暇の使用の受領と承認の確認。
- **有給バッチの更新：** 新しい有給休暇の日数が従業員のアカウントに付与された日の自動アラート。
- **契約とドキュメント：** 新しい雇用契約が発行されたり、派遣会社によって重要なドキュメントが提供されたりしたときの通知。

### 期待される結果

プラットフォームが申請のすべてのステップについて積極的に更新情報を提供し、完璧なコミュニケーションでセルフサービスサイクルを締めくくるため、従業員はサポートされ、十分な情報を提供されていると感じます。
