# 通知とアラート

従業員をバックオフィス業務と同期させるためのプロアクティブなプッシュメッセージング。

### どのような機能ですか？

**WanSpot**にネイティブ統合された集中型トランザクションアラートエンジンであり、従業員の配置に関するステータスの変更や重要なイベントについて非同期で通知するように開発されています。

### 何のためのものですか？

不安に駆られた運用サポートへの問い合わせ（「前借りは承認されたか？」、「休暇は登録されたか？」など）を根絶するためです。派遣会社によってアクションが検証され次第、システムが従業員に手続きに関する情報を継続的に提供する役割を担います。

### どのように使われますか？

アラートはシステムによって自動的にトリガーされ、以下に関する即時通知を提供します。

- **給与前借：** 財務部門による承認または拒否の即時フィードバック。
- **欠勤および有給休暇：** 欠勤記録および有給休暇の使用に関するプロトコルの確認と決定。
- **有給休暇残日数の更新：** 法定の年次有給休暇が付与された際の自動トリガー。
- **契約書とドキュメント：** 更新された契約書、重要なアナウンス、または税務ファイルが従業員の書類に追加された際の強制的な通知。

### 期待される結果

人事チームの業務負荷が大幅に軽減されます。透明性があり、監査可能で瞬時のコミュニケーションにより、セルフサービスサイクルが運用上の完全性に達し、従業員にとっての予測可能性が確保されます。
