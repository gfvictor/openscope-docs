# デジタル給与明細

経理のデジタル化と給与明細の自動配信。

### どのような機能ですか？

月次給与明細の保管と構造化された提示に焦点を当てた、セルフサービスの財務モジュールです。

### 何のためのものですか？

人事および財務部門の物流上のオーバーヘッドを根絶するためです。工場での大量の印刷、封入、対面での手渡し（または手動でのPDF送信）を、自動化された給与明細提供システムに置き換えます。

### どのように使われますか？

経理部門が**Rakudash**で給与計算を締めるとすぐに、給与明細が**WanSpot**で即座に利用可能になります。**ファイル**メニューにアクセスし、**給与明細**オプションを選択することで、従業員は対象月を選択して労働時間、手当、控除の完全な内訳を表示し、個人の記録用にいつでもPDFドキュメントをダウンロードできます。

### 期待される結果

給与明細配布の完全な自動化。従来、印刷と物流に数日を費やしていたプロセスが、絶対的な追跡可能性とデータプライバシー法の遵守を伴って、即座に行われるようになります。
