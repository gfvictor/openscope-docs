# デジタル給与明細

日本の従業員の経済生活は、予測可能性と組織性に基づいています。**WanSpot**は支払いに対する絶対的なコントロールと透明性を提供します。

### これは何ですか？

毎月の労働に関する財務ファイルとデータを閲覧することに特化したポータルエリアです。

### 何のためにあるのですか？

従来のモデルでは、人事部が何十枚もの給与明細を印刷して工場に届けるか、チャット経由でPDFを1つずつ生成して送信する時間を無駄にしていました。このモジュールはこのプロセスを自動化し、従業員が自身の給与明細に自律的にアクセスできるようにします。

### どのように使われますか？

経理部門が**Rakudash**で給与計算を締めるとすぐに、給与明細が**WanSpot**の`/payrolls`タブで即座に公開されます。従業員は希望する月をクリックして、労働時間、手当、控除の内訳を確認し、いつでもPDFドキュメントをダウンロードできます。

### 期待される結果

給与明細の配布の完全な分散化。以前は印刷、整理、配達に数日かかっていたプロセスが、コストゼロかつ完全なデータセキュリティで瞬時に行われるようになります。
