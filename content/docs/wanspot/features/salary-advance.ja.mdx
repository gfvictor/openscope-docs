# 給与前払い（前借り）

財務部門の業務を煩雑にすることなく、必要なときに従業員の流動性を確保します。

### これは何ですか？

公式の給料日前に、従業員が自分自身の給与から引き出しや前払い（いわゆる「前借り」）を申請するための統合セルフサービスシステムです。

### 何のためにあるのですか？

日本の人材派遣会社において最も繰り返され、混沌とするプロセスの1つを自動化するためです。WhatsAppに散在する申請は、財務管理と監査を困難にし、将来の控除でエラーを生み出します。

### どのように使われますか？

従業員は`/salary-advance`ルートにアクセスし、前借りしたい金額（派遣会社が定めた限度額を遵守）を入力して申請を送信します。申請は構造化された形で**Rakudash**の管理ダッシュボードに届き、人事部が評価、承認、または拒否します。

### 期待される結果

前借りの申請が集中化され、整理され、追跡可能になり、処理がはるかに迅速になります。派遣会社は送金されたすべての円の監査可能な履歴を保持し、控除は給与計算で自動的にスケジュールされます。
