# 給与前借

給与計算とネイティブに統合されたオンデマンドの流動性コントロール。

### どのような機能ですか？

従業員が、事前に確立されたビジネスルールと運用上の上限に従って給与の前借りを申請できるようにする、財務セルフサービススイートです。

### 何のためのものですか？

労働信用の付与プロセスにおける非公式性と断片化を根絶するためです。メッセージングアプリを介した申請を排除し、不適切な控除や現金不足による負債から財務部門を保護します。

### どのように使われますか？

従業員は**給与前借**タブにアクセスし、希望する引き出し金額（派遣会社が定めた運用上限を厳格に遵守）を入力して申請を送信します。申請は構造化された形で**Rakudash**の管理ダッシュボードにルーティングされ、財務部門による評価、承認、または拒否を待ちます。

### 期待される結果

エンドツーエンドで追跡可能および監査可能な財務フローの統合。プロセスは規模と会計上の厳格さを獲得し、控除は**Rakudash**の給与計算の締め切り時に自動的に引当されます。
